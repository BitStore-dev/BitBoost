# Discord Nitro Token Checker, forked from BitBoost
Tool that sorts discord tokens to see if they have nitro or not...

# How to use
1. Run ```setup.bat```
2. Paste your tokens into ```tokens.txt```
3. Run ```nitro-token-checker.py```
